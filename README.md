# pysysmon

A Python script that reads the following system parameters using psutil

- disk(s) usage %
- CPU(s) usage %
- memory usage %
- CPU temp in Â°C

Then generates an HTML file with a table listing the parameters/values

## Requirements

- Python3 
- psutil

Install using pip or apt:

```pip install psutil```

or

```apt install python3-psutil```
